# 📄 ViT-FIQA Evaluation on Yale Dataset

## 📌 Overview

This project reproduces the evaluation pipeline from the paper:

**"ViT-FIQA: Assessing Face Image Quality using Vision Transformers"**

We apply the same methodology on the **Cropped Yale Face Dataset** and generate:

- Face image quality scores using ViT-FIQA  
- Error vs Discard Curve (EDC)  
- AUC value (as reported in the paper)  

---

## 🎯 Objective

The goal is to evaluate how well a **Face Image Quality Assessment (FIQA)** model improves face recognition by:

1. Predicting quality scores for images  
2. Removing low-quality samples  
3. Measuring improvement in recognition performance
